# CrackDict
## hashCat -- CrackDict 中文拼音wifi密码字典生成工具 Python3 以上适用
### 基本功能,根据设定规则,生成字典文件适用与数字密码、英文密码、汉语拼音密码等，是hashcat的掩码功能拓展

## Install

```
chmod +x cdlis.command
ln -s ./cdlis.command /usr/local/bin/cdlis
```
